Package pl.wendigo.chrome.api.media

Contains DevTools Protocol Media domain implementation accessible via MediaOperations class.

See: Description

  • Class Summary 
    Class Description
    MediaOperationsThis domain allows detailed inspection of media elementsThis API is marked as experimental in protocol definition and can change in the future.
    PlayerPropertiesChangedEventThis can be called multiple times, and can be used to set / override / remove player properties.
    PlayerEventsAddedEventSend events as a list, allowing them to be batched on the browser for less congestion.
    PlayerMessagesLoggedEventSend a list of any messages that need to be delivered.
    PlayerErrorsRaisedEventSend a list of any errors that need to be delivered.
    PlayersCreatedEventCalled whenever a player is created, or when a new agent joins and recieves a list of active players.
    PlayerMessageHave one type per entry in MediaLogRecord::Type Corresponds to kMessage
    PlayerPropertyCorresponds to kMediaPropertyChange
    PlayerEventCorresponds to kMediaEventTriggered
    PlayerErrorCorresponds to kMediaError
